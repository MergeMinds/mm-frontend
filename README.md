## Requirements
- node.js &geqslant; 18.18
- git

## Setting up
To set up project use:
```bash
git clone git@github.com:MergeMinds/mm-frontend.git mm-frontend
cd mm-frontend
npm install
npm run dev
```
Then, go to `http://localhost:8000/`. Congratulations!
